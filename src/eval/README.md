# Evaluation Module

This module contains the reusable evaluation logic behind GeoMetricLab retrieval benchmarks.

## Scope

- Descriptor comparison and ranking evaluation
- Shared metric computation utilities for validation and testing
- Lightweight evaluation helpers reused by tracked scripts

## Public focus

The public repository emphasizes evaluation paths used by GL3D and University-1652 experiments.

## Key file

- `evaluate.py`: core evaluation helpers for retrieval-style benchmarks
